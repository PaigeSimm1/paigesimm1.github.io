Summer 2023 - Sugar Crush

Built using Roblox Studio, Lua, and Blender.


Planning Sugar Crush was my senior project in high school, and I began building it in my spare time before my first semester at Boise State. Developed in Roblox Studio using Lua, it’s a candy-themed mining simulator featuring a custom 12 × 12 mining grid that generates an array of physical blocks and randomizes ore types based on depth. All code, UI, and models were created entirely by me.

Fall 2025 - Chord Generator

Built using HTML, CSS, and JavaScript.


This Chord Progression Generator is a full-stack web app I built using HTML, CSS, Bootstrap, Node.js, and the OpenAI API. Users can select a musical style, scale, time signature, and BPM, and the server generates a machine-readable chord progression. I integrated Tone.js to play back the chords in real time and designed a streamlined UI with dropdown controls, a custom loader animation, and local storage support. This project combined my backgrounds in software development and music, and taught me a lot about API integration, asynchronous workflows, and structuring clean, interactive web tools.

// Protects against playing a blank progression
async function playProgression() {
    if (!chordProgression.length) return alert ("Generate a chord progression first!");

    await Tone.start(); // Needed to start the audio
    const synth = new Tone.PolySynth(Tone.Synth).toDestination(); // Starts the Tonal API's synth

    let timeOffset = 0; // Bypass since I had chords that would play over each other

    chordProgression.forEach(({ chord, duration }) => {
        const notes = Tonal.Chord.get(chord).notes.map(n => n + "4");
        Tone.Transport.scheduleOnce(time => { // Another bypass for that
            synth.triggerAttackRelease(notes, duration, time);
        }, `+${timeOffset}`);

        const durSeconds = { "1n":4, "2n":2, "4n":1, "8n":0.5 }[duration] || 1;
        timeOffset += durSeconds;
    })

    Tone.Transport.start(); // Plays the progression!
}
